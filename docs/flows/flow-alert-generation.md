# Flow: Smart Alert Engine

1. Portfolio data synced
2. Market data fetched
3. Alert engine runs rule checks:
    - Price movement
    - Volatility spike
    - Allocation drift
    - Macro triggers
4. Relevant alerts identified
5. Alerts grouped into bundles
6. Impact score calculated
7. “Why this matters” explanation generated
8. Alerts sorted by priority
9. Delivered to user (mobile/desktop)
10. User feedback improves relevance model
