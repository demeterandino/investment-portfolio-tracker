# UX Documentation

This folder contains all user experience artifacts for the Investment Portfolio Tracker project.
It showcases the UX thinking behind the features, including user flows, early wireframes,
and research insights.

## Structure

- **wireframes/** – Low-fidelity layout drafts of the key product screens
- **user-flows/** – Step-by-step flows showing how users complete core actions
- **research/** – UX findings, persona summaries, and high-level insights
