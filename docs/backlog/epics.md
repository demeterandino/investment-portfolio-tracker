# Epics — Investment Portfolio Tracker (Automation Edition)

This document outlines the main Epics of the project.  
Each Epic corresponds directly to a Feature and will later be broken down into User Stories and Issues.

## EPIC 1 — Data Import & Auto Categorization
Enable users to quickly import and correctly categorize their entire portfolio with minimal manual effort.

**Includes:**
- CSV/API import
- Auto-categorization engine
- Confidence scoring
- Manual overrides
- Error handling and validation

**Linked Issue:**  
→ [auto-categorization.md](https://github.com/demeterandino/investment-portfolio-tracker/blob/main/docs/prds-features/auto-categorization.md)

## EPIC 2 — Auto Alerts Engine
Deliver highly relevant alerts about portfolio drift, volatility, price shifts, and macro events—ranked by importance.

**Includes:**
- Alert rules engine
- Alert prioritization
- Alert bundles
- “Why this matters” explanations
- User alert settings

**Linked Issue:**  
→ [auto-alerts.md](https://github.com/demeterandino/investment-portfolio-tracker/blob/main/docs/prds-features/auto-alerts.md)

## EPIC 3 — Automated Rebalancing Engine
Automatically detect portfolio drift and propose suggested trades with clear, transparent logic.

**Includes:**
- Drift detection
- Rebalance recommendations
- Impact simulation
- User action workflows
- Rebalance logs

## EPIC 4 — Portfolio Analytics & Dashboard
Provide a clear and meaningful overview of portfolio allocation, risk, diversification, and trends.

**Includes:**
- Allocation analytics
- Risk scoring
- Diversification heatmap
- Overweight/underweight indicators
- Portfolio health score

## EPIC 5 — Monthly Automated Report
Generate a monthly summary of automated actions, insights, improvements, and portfolio stability metrics.

**Includes:**
- Automation summary
- Trend changes
- Top alerts of the month
- Downloadable PDF report


## EPIC 6 — User Settings & Personalization
Allow users to customize risk levels, thresholds, alert frequency, and diversification goals.

**Includes:**
- Risk profile settings
- Alert preferences
- Threshold controls
- Portfolio goals
